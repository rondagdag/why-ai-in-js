Client-Side AI Demos

A collection of demos showcasing client-side AI capabilities.

  • TensorFlow.js - Toxicity Detection

    Real-time text toxicity detection using TensorFlow.js models in the browser.

  • ONNX Runtime Web - Simple Demo

    Run ONNX models directly in the browser using both ESM and traditional script tag approaches.

  • ONNX Runtime Web - Phi Chat

    Interactive chat interface using Phi model with ONNX Runtime Web for efficient browser-based inference.

  • MediaPipe - LLM Demo

    Interactive machine learning using MediaPipe with Gemma 2B model for efficient client-side inference.

  • WebLLM - Simple Chat

    A lightweight chat interface demonstrating browser-based large language models.

  • Transformer.js - Video Object Detection

    Live object detection and tracking in video streams using browser-based ML models.

  • Transformer.js -Video Background Removal

    Real-time video background removal using ML models running entirely in the browser.

  • Transformer.js - Phi 3.5 WebGPU Demo

    WebGPU-powered Phi-3.5 language model running directly in the browser.

  • Built-In AI - Prompt API Playground

    Interactive environment to experiment with various prompt APIs and test different AI query patterns.

  • Built-In AI - Summarization API Playground

    Test environment for text summarization algorithms with real-time API integration.

  • Built-In AI - Translation & Language Detection API

    Comprehensive testing ground for translation and language detection capabilities.

  • Built-In AI Chrome Extension - Explain in Generations

    Browser extension that breaks down complex topics into different levels of understanding.

  • Built-In AI Chrome Extension - Techstack Time Machine

    Browser extension that breaks down complex topics into relatable tech stack.

Start the server locally using npx http-server in this directory.
